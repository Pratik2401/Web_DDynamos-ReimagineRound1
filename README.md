# 🌀 ReImagine Round 1 - Web_DDynamos

A creative, interactive, and responsive web project built for **ReImagine Hackathon (Round 1)** by Team DDynamos. This project demonstrates front-end development skills through a dynamic single-page portfolio and product showcase website.

---

## 🧑‍💻 Project Overview

This project is a visually engaging, personal portfolio-style and product landing website, showcasing frontend design capabilities using **HTML**, **CSS**, and **JavaScript**.  
It features smooth transitions, scroll effects, and interactive elements—reflecting a blend of aesthetics and functionality.

---

## 🌟 Features

- 🎨 Beautiful, modern UI with smooth scrolling and section-based navigation
- 🧩 Modular file structure: clean separation of HTML, CSS, and JS
- ⚙️ Responsive design: optimized for desktops and tablets
- 🖱️ Mouse-based parallax effect on the hero section
- 🧠 Multiple custom-designed sections:
  - Home / Hero
  - About
  - Skills
  - Projects / Products
  - Contact
- 🛍️ Product showcase sections (e.g., car charger, sleeves, laptop stand, etc.)
- 🖼️ Animated carousel, video backgrounds, and interactive cards

## 🚀 Getting Started

To run this project locally:

1. Clone the repository  
   ```bash
   git clone https://github.com/Pratik2401/Web_DDynamos-ReimagineRound1.git
   ```
2. Open the folder and launch `index.html` in your browser.

No server setup or dependencies required. Purely static.
