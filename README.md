# 🌀 ReImagine Round 1 - Web_DDynamos

A creative, interactive, and responsive web project built for **ReImagine Hackathon (Round 1)** by Team DDynamos. This project demonstrates front-end development skills through a dynamic single-page portfolio website.

## 🧑‍💻 Project Overview

This project is a visually engaging personal portfolio-style website, showcasing frontend design capabilities using **HTML**, **CSS**, and **JavaScript**. It contains smooth transitions, scroll effects, and interactive elements, reflecting a blend of aesthetics and functionality.

## 🌟 Features

- 🎨 Beautiful, modern UI with smooth scrolling and section-based navigation
- 🧩 Modular file structure: clean separation of HTML, CSS, and JS
- ⚙️ Responsive design: optimized for desktops and tablets
- 🖱️ Mouse-based parallax effect on the hero section
- 🧠 Multiple custom-designed sections:
  - Home / Hero
  - About
  - Skills
  - Projects
  - Contact
